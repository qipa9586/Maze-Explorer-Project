# 迷宫课设疑问

## 1. Cell 结构体墙 bool vs 越界检查

两回事，别混：
- **墙的 bool（top/right/bottom/left）** — 格子数据，表示"这面墙在不在"。true = 有墙挡着；false = 墙拆了，能过去。
- **越界检查** — 代码逻辑，`if (nextRow < 0 || nextRow >= rows || ...) continue;`，跟结构体字段无关。

迷宫外缘靠外面的墙永远不会拆（没有邻居），天然就是外墙。bool 管数据，越界检查管安全。

## 2. 为什么需要偏移数组

四个方向的邻居坐标，不用偏移数组要写四段几乎一样的代码。偏移数组把"行/列变化量"抽出来放进 `dr[]` 和 `dc[]`，一个 for 循环搞定四个方向。相同逻辑不复制粘贴。

## 3. srand(time(NULL)) 放哪

放 main() 开头，只播一次种。放函数里短期多次调用（比如按 R 重新生成），time(NULL) 返回相同秒，随机序列相同。
